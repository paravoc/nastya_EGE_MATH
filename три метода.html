Метод производной

Используется для исследования функций, зависящих от параметра, на экстремумы, монотонность и поведение.

  • Шаг 1: Найти производную функции
  • Шаг 2: Приравнять производную к нулю
  • Шаг 3: Исследовать знак производной
  • Шаг 4: Определить влияние параметра
  • Шаг 5: Найти условия для параметра

Применение: задачи на нахождение количества корней, экстремумы, наибольшие/наименьшие значения.

Аналитический метод

Основан на алгебраических преобразованиях и логических рассуждениях для нахождения значений параметра.

  • Шаг 1: Записать условие задачи в виде уравнения
  • Шаг 2: Выразить параметр через переменную
  • Шаг 3: Проанализировать область значений
  • Шаг 4: Решить полученные уравнения/неравенства
  • Шаг 5: Отобрать подходящие значения параметра

Применение: задачи на расположение корней, симметричные уравнения, системы уравнений.

Графический метод

Основан на визуализации: строятся графики функций и анализируется их взаимное расположение в зависимости от параметра.

  • Шаг 1: Записать уравнение в виде f(x) = g(x, a)
  • Шаг 2: Построить график одной части уравнения
  • Шаг 3: Исследовать семейство графиков второй части
  • Шаг 4: Определить количество точек пересечения
  • Шаг 5: Найти условия на параметр

Применение: задачи с параметром под знаком модуля, уравнения с обратными функциями.

Пример задачи с параметром

Задача:

При каких значениях параметра a уравнение |x² - 4x + 3| = a имеет ровно три корня?

Решение методом производной:

  1. Рассмотрим функцию f(x) = |x² - 4x + 3|
  2. Найдем производную на каждом интервале:
    • При x² - 4x + 3 ≥ 0 (x ≤ 1 или x ≥ 3): f(x) = x² - 4x + 3, f'(x) = 2x - 4
    • При x² - 4x + 3 < 0 (1 < x < 3): f(x) = -x² + 4x - 3, f'(x) = -2x + 4
  3. Критические точки: x = 1, x = 2, x = 3 (точки, где производная равна 0 или не существует)
  4. Вычислим значения функции в критических точках:
    • f(1) = |1 - 4 + 3| = 0
    • f(2) = |4 - 8 + 3| = 1
    • f(3) = |9 - 12 + 3| = 0
  5. График функции имеет форму "горки" с максимумом f(2)=1 и минимумами f(1)=0, f(3)=0
  6. Уравнение |x² - 4x + 3| = a имеет ровно три решения когда прямая y=a пересекает график в трех точках
  7. Это происходит при a = 1

Ответ: a = 1

Решение аналитическим методом:

  1. Исходное уравнение: |x² - 4x + 3| = a
  2. Раскроем модуль:
    • Случай 1: x² - 4x + 3 = a, при условии x² - 4x + 3 ≥ 0
    • Случай 2: -x² + 4x - 3 = a, при условии x² - 4x + 3 < 0
  3. Рассмотрим каждый случай отдельно:
    • Для первого случая: x² - 4x + (3 - a) = 0
    • Дискриминант: D₁ = 16 - 4(3 - a) = 4 + 4a = 4(1 + a)
    • Для второго случая: x² - 4x + (3 + a) = 0
    • Дискриминант: D₂ = 16 - 4(3 + a) = 4 - 4a = 4(1 - a)
  4. Анализируем количество корней в зависимости от a:
    • При a < 0: решений нет, так как модуль не может быть отрицательным
    • При a = 0: уравнение x² - 4x + 3 = 0 имеет два корня: x=1 и x=3
    • При 0 < a < 1: уравнение имеет 4 корня (по два в каждом случае)
    • При a = 1: D₁ = 8 > 0 (два корня), D₂ = 0 (один корень) - всего 3 корня
    • При a > 1: D₁ > 0 (два корня), D₂ < 0 (нет корней) - всего 2 корня

Ответ: a = 1

Решение графическим методом:

  1. Построим график функции y = |x² - 4x + 3|
  2. Для этого сначала построим параболу y = x² - 4x + 3 = (x-1)(x-3)
  3. Отразим часть параболы, находящуюся ниже оси OX, симметрично относительно этой оси
  4. Получим график в форме "горки" с вершинами в точках (1,0), (2,1), (3,0)
  5. Уравнение |x² - 4x + 3| = a означает пересечение этого графика с горизонтальной прямой y = a
  6. Количество корней уравнения равно количеству точек пересечения
  7. Проанализируем:
    • При a < 0: нет пересечений (0 корней)
    • При a = 0: две точки пересечения (2 корня)
    • При 0 < a < 1: четыре точки пересечения (4 корня)
    • При a = 1: три точки пересечения (3 корня)
    • При a > 1: две точки пересечения (2 корня)

Ответ: a = 1

Графическая иллюстрация:

Изменяйте параметр a и наблюдайте, как меняется количество точек пересечения (корней уравнения).

Выводы:

Все три метода привели к одному и тому же ответу: a = 1. Выбор метода зависит от конкретной задачи:

  • Производная хороша для исследования функций на экстремумы и монотонность
  • Аналитический метод надежен и систематичен, но может быть громоздким
  • Графический метод наиболее нагляден и часто позволяет быстро получить ответ

При подготовке к ЕГЭ рекомендуется освоить все три метода, чтобы уметь выбирать оптимальный для каждой задачи.
