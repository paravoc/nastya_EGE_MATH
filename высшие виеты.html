📐 I. Теорема Виета (общая форма для многочлена n-й степени)

Пусть дан многочлен степени \(n\):

$$P(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0, \quad a_n \neq 0$$

и пусть \(x_1, x_2, \dots, x_n\) — его корни (действительные или комплексные, с учётом кратности). Тогда выполняются следующие соотношения (формулы Виета):

🔷 Общая система Виета

\(x_1 + x_2 + \dots + x_n = -\dfrac{a_{n-1}}{a_n}\)

\(\displaystyle\sum_{1 \le i < j \le n} x_i x_j = \dfrac{a_{n-2}}{a_n}\)

\(\displaystyle\sum_{1 \le i < j < k \le n} x_i x_j x_k = -\dfrac{a_{n-3}}{a_n}\)

\(\vdots\)

\(x_1 x_2 \cdots x_n = (-1)^n \dfrac{a_0}{a_n}\)

правило знаков Знак чередуется: сумма (1 корень) — минус, сумма парных произведений — плюс, тройных — минус, и так далее. Последний знак: \((-1)^n\).

📌 Частные случаи (шпаргалка)

Квадратное (\(n=2\)):
\(x_1 + x_2 = -\frac{b}{a}\)
\(x_1 x_2 = \frac{c}{a}\)
Кубическое (\(n=3\)):
\(x_1+x_2+x_3 = -\frac{b}{a}\)
\(x_1x_2 + x_1x_3 + x_2x_3 = \frac{c}{a}\)
\(x_1x_2x_3 = -\frac{d}{a}\)

🧩 Применение: выражение симметрических многочленов

\(x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2\)
\(x_1^3 + x_2^3 = (x_1+x_2)^3 - 3x_1x_2(x_1+x_2)\)
\(\displaystyle\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1+x_2}{x_1x_2}\)
\(|x_1 - x_2| = \frac{\sqrt{D}}{|a|}\)
Пример (кубическое уравнение)

Условие: Для уравнения \(x^3 - 6x^2 + 11x - 6 = 0\) найти \(x_1^2 + x_2^2 + x_3^2\).

Решение:

  1. По Виету: \(S_1 = x_1+x_2+x_3 = 6\), \(S_2 = x_1x_2+x_1x_3+x_2x_3 = 11\).
  2. Используем тождество: \((x_1+x_2+x_3)^2 = x_1^2+x_2^2+x_3^2 + 2S_2\).
  3. Подставляем: \(36 = x_1^2+x_2^2+x_3^2 + 22\).
  4. Отсюда: \(x_1^2+x_2^2+x_3^2 = 14\).

Ответ: 14.

⚡ II. Уравнения высших порядков (n ≥ 3)

🔍 1. Теорема о рациональных корнях (основной инструмент)

📖 Теорема

Если многочлен \(a_n x^n + a_{n-1}x^{n-1} + \dots + a_0\) с целыми коэффициентами имеет рациональный корень \(\frac{p}{q}\) (несократимая дробь), то:

  • \(p\) — делитель свободного члена \(a_0\);
  • \(q\) — делитель старшего коэффициента \(a_n\).

📋 Алгоритм поиска рациональных корней

  1. Выписать все положительные и отрицательные делители \(a_0\).
  2. Выписать все положительные делители \(a_n\).
  3. Составить все возможные дроби \(\frac{\text{делитель } a_0}{\text{делитель } a_n}\).
  4. Подставлять каждую дробь в уравнение, пока не получится 0.
  5. Если \(a_n = 1\), кандидаты — просто целые делители \(a_0\).
Пример (рациональные корни)

Уравнение: \(2x^3 - 3x^2 - 11x + 6 = 0\)

Решение:

  • \(a_0 = 6\), делители: \(\pm 1, \pm 2, \pm 3, \pm 6\).
  • \(a_n = 2\), делители: \(\pm 1, \pm 2\).
  • Кандидаты: \(\pm 1, \pm 2, \pm 3, \pm 6, \pm \frac12, \pm \frac32\).
  • Проверяем \(x=3\): \(54 - 27 - 33 + 6 = 0\) → корень.

Далее делим на \(x-3\) (Горнер) и решаем квадратное.

Ответ: \(3,\; \frac12,\; -2\).

🧮 2. Схема Горнера (быстрое деление многочлена)

Позволяет разделить многочлен на \((x - c)\) и одновременно получить коэффициенты частного и остаток.

Коэффициенты: \(a_n, a_{n-1}, ..., a_0\)
c → ↓
---
\(b_n = a_n\)
\(b_{n-1} = a_{n-1} + c \cdot b_n\)
\(b_{n-2} = a_{n-2} + c \cdot b_{n-1}\)
...
Остаток = \(a_0 + c \cdot b_1\)

Пример: Разделить \(x^4 - 4x^3 - x^2 + 16x - 12\) на \(x-2\).

Коэфф: 1, -4, -1, 16, -12; c = 2.

1-4-116-12
2↓12-4-1012
1-2-560

Частное: \(x^3 - 2x^2 - 5x + 6\), остаток 0.

🔄 3. Специальные замены (понижение степени)

Биквадратное:
\(ax^4 + bx^2 + c = 0\)
замена \(t = x^2 \ge 0\)
Трехчленное:
\(ax^{2n} + bx^n + c = 0\)
замена \(t = x^n\)
Пример (трехчленное)

\(x^6 - 9x^3 + 8 = 0\)

Замена \(t = x^3\): \(t^2 - 9t + 8 = 0\) → \(t = 1,\; 8\).

\(x^3 = 1\) → \(x = 1\) (и два комплексных).

\(x^3 = 8\) → \(x = 2\) (и два комплексных).

Действительные корни: \(1,\; 2\).

🔄 III. Симметрические (возвратные) уравнения

📌 Определение

Уравнение называется возвратным (симметрическим), если его коэффициенты, равноудалённые от концов, равны между собой.

Чётная степень (4,6,8):
\(a_n = a_0,\; a_{n-1} = a_1,\; a_{n-2} = a_2,\dots\)
Нечётная степень (5,7,9):
\(a_n = a_0,\; a_{n-1} = a_1,\; a_{n-2} = a_2,\dots\)

🎯 1. Уравнения 4-й степени (чётная)

Вид: \(ax^4 + bx^3 + cx^2 + bx + a = 0\)

Алгоритм:

  1. Убедиться, что \(x=0\) не корень (иначе вынести).
  2. Разделить обе части на \(x^2\).
  3. Сгруппировать: \(a\left(x^2 + \frac{1}{x^2}\right) + b\left(x + \frac{1}{x}\right) + c = 0\).
  4. Замена \(t = x + \frac{1}{x}\). Тогда \(x^2 + \frac{1}{x^2} = t^2 - 2\).
  5. Получаем квадратное: \(a t^2 + b t + (c - 2a) = 0\).
  6. Решаем относительно \(t\), затем возвращаемся к \(x\): \(x^2 - t x + 1 = 0\).
Пример (4-я степень)

Уравнение: \(x^4 - 5x^3 + 6x^2 - 5x + 1 = 0\)

  1. Делим на \(x^2\): \(x^2 - 5x + 6 - \frac{5}{x} + \frac{1}{x^2} = 0\).
  2. Группируем: \((x^2 + \frac{1}{x^2}) - 5(x + \frac{1}{x}) + 6 = 0\).
  3. Замена \(t = x + \frac{1}{x}\) → \(t^2 - 2 - 5t + 6 = 0\) → \(t^2 - 5t + 4 = 0\).
  4. \(t = 1\) или \(t = 4\).
  5. \(x + \frac{1}{x} = 1\) → \(x^2 - x + 1 = 0\) (нет решений).
  6. \(x + \frac{1}{x} = 4\) → \(x^2 - 4x + 1 = 0\) → \(x = 2 \pm \sqrt{3}\).

Ответ: \(2 \pm \sqrt{3}\).

⚡ 2. Уравнения 5-й степени (нечётная)

Вид: \(ax^5 + bx^4 + cx^3 + cx^2 + bx + a = 0\)

ключевой факт Всегда есть корень \(x = -1\) (проверяется подстановкой).

Алгоритм:

  1. Проверить \(x = -1\) (если да — корень). Если знаки чередуются (плюс-минус-плюс...), то корень \(x = 1\).
  2. Разделить многочлен на \(x+1\) (или \(x-1\)) по схеме Горнера.
  3. Полученное частное — уравнение 4-й степени. Решить его методом для чётных возвратных.
Пример (5-я степень)

Уравнение: \(x^5 + 2x^4 - 3x^3 - 3x^2 + 2x + 1 = 0\)

  1. Проверяем \(x = -1\): \(-1 + 2 + 3 - 3 - 2 + 1 = 0\) → да.
  2. Делим на \(x+1\) (Горнер, c=-1): получаем \(x^4 + x^3 - 4x^2 + x + 1 = 0\).
  3. Решаем это уравнение 4-й степени (делим на \(x^2\), замена \(t = x + 1/x\)):
    • \(x^2 + x - 4 + \frac{1}{x} + \frac{1}{x^2} = 0\) → \((x^2 + \frac{1}{x^2}) + (x + \frac{1}{x}) - 4 = 0\).
    • \(t^2 - 2 + t - 4 = 0\) → \(t^2 + t - 6 = 0\) → \(t = 2\) или \(t = -3\).
    • \(x + 1/x = 2\) → \(x = 1\).
    • \(x + 1/x = -3\) → \(x^2 + 3x + 1 = 0\) → \(x = \frac{-3 \pm \sqrt{5}}{2}\).
  4. Не забываем корень \(x = -1\) из первого шага.

Ответ: \(-1,\; 1,\; \frac{-3 \pm \sqrt{5}}{2}\).

🌀 3. Обобщённая замена \(t = x + \frac{m}{x}\)

Если уравнение 4-й степени имеет вид \(ax^4 + bx^3 + cx^2 + dx + e = 0\) и \(\frac{e}{a} = \left(\frac{d}{b}\right)^2\), то после деления на \(x^2\) применяют замену \(t = x + \frac{m}{x}\), где \(m = \sqrt{\frac{e}{a}}\).

Пример (обобщённый)

Уравнение: \(x^4 - 4x^3 - 2x^2 - 4x + 1 = 0\)

  1. Делим на \(x^2\): \(x^2 - 4x - 2 - \frac{4}{x} + \frac{1}{x^2} = 0\).
  2. Группируем: \((x^2 + \frac{1}{x^2}) - 4(x + \frac{1}{x}) - 2 = 0\).
  3. Замена \(t = x + \frac{1}{x}\) → \(t^2 - 2 - 4t - 2 = 0\) → \(t^2 - 4t - 4 = 0\).
  4. \(t = 2 \pm 2\sqrt{2}\).
  5. Возвращаемся: \(x^2 - (2 \pm 2\sqrt{2})x + 1 = 0\).

Корни получаются громоздкими, но метод работает.

🚀 Итоговая стратегия для любого уравнения:
1. Проверить на симметрию (возвратность) — применить замену \(t = x \pm 1/x\).
2. Если не симметрично — искать рациональные корни (теорема о делителях).
3. Найденный корень отсекать схемой Горнера, понижая степень.
4. Повторять, пока не сведётся к квадратному или биквадратному.
