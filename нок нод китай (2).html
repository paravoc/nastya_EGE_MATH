🍡 китайский остаточек

основная формула:

\[ x \equiv a \pmod{m} \quad\text{и}\quad x \equiv b \pmod{n} \]
все решения: \(x = x_0 + k \cdot \text{НОК}(m, n)\)
🍬 задача 1

Найдите наименьшее натуральное число, которое при делении на 5 даёт остаток 3, а при делении на 7 даёт остаток 4.

решение: числа с остатком 3 при делении на 5: 3, 8, 13, 18, 23… Проверяем остатки при делении на 7: 3→3, 8→1, 13→6, 18→4. Значит, \(x_0 = 18\). НОК(5,7)=35, поэтому все числа: \(18 + 35k\). Наименьшее — 18.

🍬 задача 2

Найдите все трёхзначные числа, которые при делении на 5 дают остаток 3, а при делении на 7 — остаток 4.

решение: из предыдущей задачи общий вид \(x = 18 + 35k\). Трёхзначные: \(100 \le 18+35k \le 999\). \(k\) от 3 до 28. Получаем 26 чисел.

🍬 задача 3

Разложите \((2x - 3)^4\) по формуле бинома.

решение: \((2x)^4 + 4(2x)^3(-3) + 6(2x)^2(-3)^2 + 4(2x)(-3)^3 + (-3)^4 = 16x^4 - 96x^3 + 216x^2 - 216x + 81\).

🍰 степени: aⁿ + bⁿ и aⁿ – bⁿ

🍬 aⁿ + bⁿ (n нечётное)

\[ a^n + b^n = (a+b)(a^{n-1} - a^{n-2}b + a^{n-3}b^2 - \dots + b^{n-1}) \]
🍬 задача 4

Разложите на множители \(x^5 + 32\).

решение: \(x^5 + 2^5 = (x+2)(x^4 - 2x^3 + 4x^2 - 8x + 16)\).

🧁 aⁿ – bⁿ (любое n)

\[ a^n - b^n = (a-b)(a^{n-1} + a^{n-2}b + a^{n-3}b^2 + \dots + ab^{n-2} + b^{n-1}) \]
🍬 задача 5

Сократите дробь \(\frac{x^4 - 81}{x - 3}\).

решение: по формуле для n=4: \(x^4 - 3^4 = (x-3)(x^3 + 3x^2 + 9x + 27)\). Сокращаем на \(x-3\), получаем \(x^3 + 3x^2 + 9x + 27\).

🍬 задача 6

Докажите, что \(11^{10} - 1\) делится на 10.

решение: \(11^{10} - 1^{10} = (11-1)(11^9 + 11^8 + \dots + 1) = 10 \cdot (\text{целое число})\), значит, делится на 10.

🍦 сумма квадратов 1²+2²+...+n²

\[ 1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6} \]

🫧 вывод через разность кубов

\[ (k+1)^3 - k^3 = 3k^2 + 3k + 1 \]
Суммируем от \(k=1\) до \(n\):
\[ (n+1)^3 - 1 = 3\sum k^2 + 3\sum k + n \]
\[ 3S = (n+1)^3 - 1 - 3\cdot\frac{n(n+1)}{2} - n \]
\[ S = \frac{n(n+1)(2n+1)}{6} \]
🍬 задача 7

Найдите сумму \(1^2 + 2^2 + \dots + 50^2\).

решение: \(\frac{50 \cdot 51 \cdot 101}{6} = \frac{50 \cdot 51 \cdot 101}{6} = 25 \cdot 17 \cdot 101 = 42925\).

🍬 задача 8

Сколько нужно сложить первых квадратов, чтобы сумма стала больше 1000?

решение: \(\frac{n(n+1)(2n+1)}{6} > 1000\). При n=13 сумма = 819, при n=14 сумма = 1015. Значит, нужно 14 слагаемых.

🍧 площадь под параболой y = x²

\[ \int_0^1 x^2 dx = \frac{1}{3} \]

🔗 связь с суммой квадратов

\[ S_n = \frac{1}{n^3}(1^2+2^2+\dots+n^2) = \frac{(n+1)(2n+1)}{6n^2} \]
\[ \lim_{n\to\infty} S_n = \frac{2}{6} = \frac{1}{3} \]
🍬 задача 9

Вычислите приближённое значение интеграла \(\int_0^1 x^2 dx\) с помощью суммы квадратов при n=10.

решение: \(S_{10} = \frac{11 \cdot 21}{600} = \frac{231}{600} = 0,385\). Точное значение 0,333…, погрешность ≈ 0,052.

🍬 задача 10

Найдите все общие решения для чисел, дающих остаток 2 при делении на 4 и остаток 3 при делении на 6.

решение: подбором находим \(x_0 = 3\) (3 mod 4 = 3? нет, 3 mod 4 = 3 — подходит, 3 mod 6 = 3 — тоже подходит). НОК(4,6)=12. Все числа: \(3 + 12k\).
